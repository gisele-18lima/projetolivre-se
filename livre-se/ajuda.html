COMO FAÇO PARA RECEBER UMA DOAÇÃO?

PARA RECEBER UMA DOAÇÃO VOCÊ SE CADASTRA, MOSTRA INTERESSE EM UM LIVRO E CONVERSA COM QUEM ESTÁ DISPOSTO A DOÁ-LO.

COMO FAÇO PARA SER UM DOADOR?

PARA SE TORNAR UM DOADOR VOCÊ TEM QUE ESTAR CADASTRADO NO SITE E REALIZAR O CADASTRO DOS LIVROS QUE PRETENDE DOAR.

COMO ALTERAR O MEU ENDEREÇO APÓS REALIZAR O CADASTRO?

VOCÊ IRÁ NO SEU PERFIL DE USUÁRIO, ALTERAR DADOS E MUDAR O SEU ENDEREÇO.

SOU OBRIGADO(A) A COLOCAR FOTO?

NÃO É OBRIGATÓRIO, MAS QUANTO AOS DOADORES É INTERESSANTE PARA QUE SUA FOTO ESTEJA NA ABA DE DOADORES COM INFORMAÇÕES DE QUANTOS LIVROS JÁ FORAM DOADOS E UM FEEDBACK DE DONATÁRIOS.

como saber se a pessoa que vende livro não é golpe?

o livre-se pede o cpf no cadastro, caso seja apontado um cpf inexistente o usuário não poderá se cadastrar. como nossa plataforma não lida com dinheiro, o golpe não é um risco relevante.

quantos dias demora para chegar o livro?

isto é algo muito relativo, se o doador e o donatário optarem por enviar o livro via correio, todo o processo pode ser conferido pelo rastreamento no site dos correios.

para que serve o nome social no cadastro?

esta é uma opção para pessoas trangênero, para que possam ser referidas com seus nomes sociais e se sentirem confortáveis utilizando o site.

qual a função da aba estude+?

esta aba é feita especialmente para ajudar aqueles que procuram conteúdos didáticos para estudar; selecionamos sites, canais do youtube e aplicativos que possam auxiliar no estudo de cada matéria e, caso a pessoa clique em algum dos links, será redirecionada para o site escolhido.

como funciona o sistema de frete do site?

o livre-se não trabalha diretamente com o frete dos livros, caso o doador e o donatário queiram enviar o livro via correio todo o custo do frete deverá ser conversado entre ambos e assim gerar um acordo das duas partes.

não quero mais doar o livro que cadastrei, o que faço?

você pode encontrar no seu perfil todos os livros que disponibiliza para doação, e caso desista de doar algum, pode clicar na opção de exclui-lo e ele sairá do site.
